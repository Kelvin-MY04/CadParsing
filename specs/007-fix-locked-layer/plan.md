# Implementation Plan: Fix PDF Export Failure on Locked Layers

**Branch**: `007-fix-locked-layer` | **Date**: 2026-02-28 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/007-fix-locked-layer/spec.md`

---

## Summary

PDF export fails with `eInvalidInput` whenever a DWG contains text entities on locked layers.
The root cause is that `TextColorOverride` and `TextFontOverride` attempt `GetObject(id, OpenMode.ForWrite)` on entities whose layers are locked — this corrupts the AutoCAD transaction, causing `transaction.Commit()` to throw `eInvalidInput`.

The fix introduces a new `LayerLockOverride` helper that pre-checks each entity's layer lock state, temporarily unlocks locked layers (with one diagnostic log per layer), applies the override, and restores the original lock state after export — guaranteeing the DWG is never permanently altered.

## Phase 0: Research

All research complete. See [research.md](research.md).

### Key Decisions

| Decision | Outcome |
|---|---|
| Root cause | `GetObject(ForWrite)` on locked-layer entity corrupts the transaction before exception is raised |
| Fix strategy | Pre-check `LayerTableRecord.IsLocked` before any `ForWrite` attempt; unlock if locked; restore after export |
| API | `LayerTableRecord.IsLocked` (bool read/write) + `UpgradeOpen()` for the in-transaction unlock |
| Unlock granularity | Per unique layer (not per entity); satisfies FR-003 (one log per layer) |
| Interaction between font and color unlocks | Independent dictionaries; no coordination needed (colour unlock detects font-override layers are already unlocked) |
| Test strategy | `[Ignore]` + manual steps — precedent from TextFontOverrideTests |

## Phase 1: Design

### New Class: `LayerLockOverride`

**File**: `CadParsing/Helpers/LayerLockOverride.cs`
**Namespace**: `CadParsing.Helpers`
**Access**: `internal static`

#### Method: `CollectLockedLayerIds`

```
Signature:
  ISet<ObjectId> CollectLockedLayerIds(
      Transaction transaction,
      IReadOnlyList<ObjectId> entityIds)

Responsibility:
  Open each entity ForRead; read its LayerId; open that LayerTableRecord ForRead;
  if IsLocked == true, add the layer ObjectId to the result set.

Returns:
  HashSet<ObjectId> — unique locked layer IDs referenced by the given entity list.

Errors:
  Per-entity exceptions are caught and swallowed (entity skipped silently);
  the method must not throw.
```

#### Method: `UnlockLayers`

```
Signature:
  Dictionary<ObjectId, bool> UnlockLayers(
      Transaction transaction,
      ISet<ObjectId> lockedLayerIds,
      Editor editor)

Responsibility:
  For each layer ID:
    1. Open LayerTableRecord ForRead, read IsLocked → save to dictionary.
    2. UpgradeOpen(), set IsLocked = false.
    3. Log: "[INFO] LayerLockOverride: Temporarily unlocking layer '{name}'."

Returns:
  Dictionary<ObjectId, bool> — savedLockStates (layerId → wasLocked).

Errors:
  Per-layer exceptions are caught; layer is skipped with a [WARN] message.
```

#### Method: `RestoreLayerLocks`

```
Signature:
  void RestoreLayerLocks(
      Transaction transaction,
      Dictionary<ObjectId, bool> savedLockStates,
      Editor editor)

Responsibility:
  For each entry in savedLockStates:
    Open LayerTableRecord ForWrite; set IsLocked = savedValue.

Errors:
  Per-layer exceptions are caught; layer is skipped with a [WARN] message.
  Guard: if savedLockStates is null or empty, return immediately.
```

### Changes to `ExportPdfCommand.ExportAllBorders`

**Location**: `CadParsing/Commands/ExportPdfCommand.cs`, method `ExportAllBorders`

**Current flow** (lines 107–148):
```
FindTextEntitiesOnTargetLayers → ApplyStandardFontOverride → [border loop] → RestoreOriginalTextStyles → Commit
```

**New flow**:
```
FindTextEntitiesOnTargetLayers
  → CollectLockedLayerIds          [NEW]
  → UnlockLayers                   [NEW]
  → ApplyStandardFontOverride
  → [border loop]
  → [finally] RestoreOriginalTextStyles
  → [finally] RestoreLayerLocks    [NEW]
  → Commit
```

**Pseudocode delta**:
```csharp
// After FindTextEntitiesOnTargetLayers:
ISet<ObjectId> fontLayerIds =
    LayerLockOverride.CollectLockedLayerIds(transaction, targetTextEntityIds);
Dictionary<ObjectId, bool> savedFontLayerLocks =
    LayerLockOverride.UnlockLayers(transaction, fontLayerIds, editor);

// In the finally block, after RestoreOriginalTextStyles:
LayerLockOverride.RestoreLayerLocks(transaction, savedFontLayerLocks, editor);
```
